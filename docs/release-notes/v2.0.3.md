# Release Notes – v2.0.3

## Highlights

v2.0.3 is a maintenance and packaging release focused on Linux distribution quality and project housekeeping. The Linux package icon setup was improved to ensure proper app icon visibility in desktop app menus (including KDE).

## Bug Fixes

### Linux Package Icon Handling (KDE/App Menu)

• Linux builds now use a dedicated multi-size icon set directory (`build/icons/linux`) instead of a single PNG file
• Added standard Linux icon sizes (16, 24, 32, 48, 64, 96, 128, 256, 512) for better desktop integration across DEs and package formats (including RPM)

## Changes

### Packaging / Metadata

• Updated app package description to reflect current provider support (X-Updater + iniBuilds)

### Documentation / Repository Maintenance

• Moved release notes into `docs/release-notes/`
• Moved x-plane.org posting texts into `docs/release-notes/x-plane-org/`
• Added security policy and issue templates
• Added Dependabot configuration and updated GitHub Actions dependencies
• Updated `minimatch` dev dependency

## Breaking Changes

None.

## Known Limitations

• DRM-protected product installation has not been tested end-to-end (requires DRM product ownership)
