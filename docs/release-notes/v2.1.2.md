# Release Notes – v2.1.2

## Highlights

v2.1.2 is a maintenance and security release focused on dependency updates and a refreshed npm lockfile to resolve transitive audit findings in the build toolchain.

## Bug Fixes

### Transitive Dependency Audit Findings

• Regenerated `package-lock.json` to refresh transitive dependency resolution
• Resolved open npm audit findings for transitive `lodash` and `brace-expansion` dependencies
• No direct application runtime dependencies were changed

## Changes

### Build / Dependency Maintenance

• Updated Electron from `40.6.1` to `41.1.1`
• Updated `tar` from `7.5.9` to `7.5.11`
• Updated `picomatch` from `4.0.3` to `4.0.4`
• Updated `@xmldom/xmldom` from `0.8.11` to `0.8.12`
• Refreshed the lockfile without changing declared top-level dependency versions
• Verified that the AppImage build still completes successfully after the lockfile refresh

## Breaking Changes

None. Fully compatible with v2.1.1 profiles and settings.

## Known Limitations

• DRM-protected product installation has not been tested end-to-end (requires DRM product ownership)
