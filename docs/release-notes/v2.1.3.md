# Release Notes – v2.1.3

## Highlights

v2.1.3 is a security and maintenance release that updates Electron and resolves a high-severity vulnerability in a transitive build dependency.

## Security Fixes

### @xmldom/xmldom – High Severity (CVE)

Updated transitive dependency `@xmldom/xmldom` from `0.8.12` to `0.8.13` via `npm audit fix`.

This release of `@xmldom/xmldom` addresses four security advisories:

- **GHSA-2v35-w6hq-6mfw** – Uncontrolled recursion in DOM traversal operations (DoS via deeply nested XML trees)
- **GHSA-f6ww-3ggp-fr8h** – XML injection via unvalidated `DocumentType` serialization
- **GHSA-x6wf-f3px-wcqx** – XML node injection via unvalidated processing instruction serialization
- **GHSA-j759-j44w-7fr8** – XML node injection via unvalidated comment serialization

`@xmldom/xmldom` is used exclusively by `electron-builder` (dev/build toolchain) and is not part of the application runtime. User data is not affected.

## Changes

### Build / Dependency Maintenance

- Updated Electron from `41.1.1` to `41.5.0`
- Updated transitive `@xmldom/xmldom` from `0.8.12` to `0.8.13` (security fix, see above)

## Breaking Changes

None. Fully compatible with v2.1.2 profiles and settings.

## Known Limitations

- DRM-protected product installation has not been tested end-to-end (requires DRM product ownership)
