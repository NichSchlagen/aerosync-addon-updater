## AeroSync Addon Updater v2.0.0 – iniBuilds Support Added

Hi everyone,  
v2.0.0 is a major feature release and turns AeroSync into a dual-provider updater.

You can now use AeroSync with:
- X-Updater
- iniBuilds

### What’s new in v2.0.0
- Full iniBuilds provider support (login, owned products, update flow)
- New provider selection per profile (X-Updater or iniBuilds)
- Provider-aware profile form in the UI
- iniBuilds activation key handling integrated into profile workflow
- Updated documentation and English/German UI label refinements

### Compatibility
- No breaking changes
- Existing v1.x profiles remain compatible

### Known limitation
- DRM-protected product installation is not yet tested end-to-end (requires DRM product ownership)

Full changelog: https://github.com/NichSchlagen/aerosync-addon-updater/compare/v1.5.0...v2.0.0
