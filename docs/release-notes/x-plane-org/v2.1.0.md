## AeroSync Addon Updater v2.1.0 – Usability Update (ETA + In-App Release Notes)

Hi everyone,  
v2.1.0 is a feature release focused on usability and transparency during updates.

### What’s new in v2.1.0
- Install progress now shows throughput and ETA during update installation
- New in-app release notes modal (markdown-based)
- Release notes open automatically when an app update check finds a newer version
- Documentation updates around rollback snapshots and plan-management behavior

### Compatibility
- No breaking changes
- Existing v2.0.x profiles remain compatible

### Known limitation
- DRM-protected product installation is not yet tested end-to-end (requires DRM product ownership)

Full changelog: https://github.com/NichSchlagen/aerosync-addon-updater/compare/v2.0.3...v2.1.0
