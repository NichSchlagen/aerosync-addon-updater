# Release Notes – v2.0.2


## Highlights

v2.0.2 is a small UI polish release. Download progress no longer spams the application log, and provider-irrelevant form fields are now hidden when using iniBuilds.


## Bug Fixes

### Download Progress Log Spam

• During iniBuilds package downloads, the application log now only writes a new line when the displayed percentage changes — previously every progress event (~4 per second) produced an identical log entry
• Download log lines now include the current percentage and transferred size (e.g. `DOWNLOAD iniBuilds package — 42% (518 MB / 1.23 GB)`) instead of repeating `0/3492 DOWNLOAD iniBuilds package`

### iniBuilds Form Field Visibility

• The **Release Channel** dropdown (Release / Beta / Alpha) is now hidden for iniBuilds profiles, since iniBuilds does not support release channels — it was previously always visible regardless of the selected provider


## Changes

None.


## Breaking Changes

None. Fully compatible with v2.0.0 and v2.0.1 profiles and settings.


## Known Limitations

• DRM-protected product installation has not been tested end-to-end (requires DRM product ownership)
