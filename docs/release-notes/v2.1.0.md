# Release Notes – v2.1.0

## Highlights

v2.1.0 is a feature release focused on update transparency and usability. During installation, AeroSync now shows throughput and ETA, and app updates can display in-app markdown release notes in a dedicated modal.

## New Features

### Install Throughput + ETA

• Added real-time install throughput display during update installation
• Added estimated remaining time (ETA) to make long-running installs easier to track
• UI and i18n updates in English and German for the new progress details

### In-App Release Notes Modal

• Added a markdown-based release notes modal in the app
• Release notes are shown automatically when update checks detect a newer version
• Added explicit open/close release notes interactions in the renderer UI
• Markdown/HTML rendering is sanitized before display

## Changes

### Documentation

• Updated `docs/user-guide.md` for the new release-notes and install-progress behavior
• Expanded rollback snapshot management documentation for both providers
• Updated troubleshooting and data/security docs for plan management behavior

### Dependencies / CI

• Updated Electron from `40.6.0` to `40.6.1`
• Updated GitHub Actions artifact steps (`actions/download-artifact` and `actions/upload-artifact`)

## Breaking Changes

None. Compatible with v2.0.x profiles and settings.

## Known Limitations

• DRM-protected product installation has not been tested end-to-end (requires DRM product ownership)
