# Release Notes – v2.1.1

## Highlights

v2.1.1 is a maintenance release focused on CI/build reliability for release packaging.

## Bug Fixes

### CI Build Stability

• Updated GitHub Actions Node.js runtime from `20` to `22` for release builds
• Added more robust npm fetch retry/timeout settings during `npm ci`
• Added a second `npm ci` attempt in the workflow to reduce failures from transient network issues (e.g. Electron download `socket hang up`)

## Changes

None.

## Breaking Changes

None. Fully compatible with v2.1.0 profiles and settings.

## Known Limitations

• DRM-protected product installation has not been tested end-to-end (requires DRM product ownership)
